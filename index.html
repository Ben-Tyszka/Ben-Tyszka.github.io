LifeBoard

This is the project I am currently working on with some of my friends. We wanted to make a mobile app that allows you to do things like create reminders, make to do lists, show an overview of all your social media accounts, and some other productivity/organizational things. The idea is that it can be used to show an overview of your whole life, hence the name LifeBoard. The backend server is a REST API written in C# with ASP.NET Core 2.0 and its EntityFramework technology. The client app is written in Google's Dart programing language with the cross platform Flutter app framework, as far as I can tell very few people use this framework which is kind of why I like it as I want a bit of a challenge. This project is certainly going to be very large and fairly complicated as I want to deploy the API to either AWS or Azure.
